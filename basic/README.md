# unit-test-good-practice

## Challenge Goal
 - none dependency
 - 100% coverage
 

## To run
```
$ git clone git@github.com:wahengchang/unit-test-good-practice.git
$ cd unit-test-good-practice
$ npm i
```

## Run test
```
$ npm test
```

## Run test coverage
```
$ npm test:coverage
```

## Dev single test file
```
$ ./node_modules/jest/bin/jest.js __tests__/lib/dataHandler.js --watch
```
